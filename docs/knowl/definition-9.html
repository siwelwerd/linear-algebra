Definition 2.1.2.

A vector space \(V\) is any collection of mathematical objects with associated addition \(\oplus\) and scalar multiplication \(\odot\) operations that satisfy the following properties. Let \(\vec u,\vec v,\vec w\) belong to \(V\text{,}\) and let \(a,b\) be scalar numbers.

  1. \(\vec u\oplus (\vec v\oplus \vec w)= (\vec u\oplus \vec v)\oplus \vec w\text{.}\)

  2. \(\vec u\oplus \vec v= \vec v\oplus \vec u\text{.}\)

  3. There exists some \(\vec z\) where \(\vec v\oplus \vec z=\vec v\text{.}\)

  4. There exists some \(-\vec v\) where \(\vec v\oplus (-\vec v)=\vec z\text{.}\)

  5. \(a(b\odot\vec v)=(ab)\odot\vec v\text{.}\)

  6. \(1\odot\vec v=\vec v\text{.}\)

  7. \(a\odot(\vec u\oplus \vec v)=(a\odot\vec u)\oplus(a\odot\vec v)\text{.}\)

  8. \((a+ b)\odot\vec v=(a\odot\vec v)\oplus(b\odot \vec v)\text{.}\)
