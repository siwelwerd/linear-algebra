Activity 1.1.2.

Consider the following consistent linear system.

\begin{align*} -x_1+2x_2 &= -3\\ 2x_1-4x_2 &= 6 \end{align*}
(a)

Find three different solutions for this system.

(b)

Let \(x_2=a\) where \(a\) is an arbitrary real number, then find an expression for \(x_1\) in terms of \(a\text{.}\) Use this to write the solution set \(\setBuilder { \begin{bmatrix} \unknown \\ a \end{bmatrix} }{ a \in \IR }\) for the linear system.
