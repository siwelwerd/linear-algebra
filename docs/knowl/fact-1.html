Fact 1.1.7.

All linear systems are one of the following:

  1. Consistent with one solution: its solution set contains a single vector, e.g. \(\setList{\begin{bmatrix}1\\2\\3\end{bmatrix}}\)
  2. Consistent with infinitely-many solutions: its solution set contains infinitely many vectors, e.g. \(\setBuilder { \begin{bmatrix}1\\2-3a\\a\end{bmatrix} }{ a\in\IR }\)
  3. Inconsistent: its solution set is the empty set \(\{\}=\emptyset\)
